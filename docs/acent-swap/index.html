    Acent Swap - Acent Swap

    The migration from Acent Swap has finally arrived and this document is going to help you get started in your journey to Acent Swap.

    Let's go through the process

    1. We go to the site
    2. Check that there is liquidity to perform our swap today (10% of supply is available each day for swapping)
    3. Choose amount to migrate from Acent Swap
    4. Approve transaction and wait
    5. Finalize transaction and wait a bit more
    6. Receive Acent back
